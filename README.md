# Society MultiRoblox

A multi-account Roblox launcher built with Electron.

## Features

- Multi-account Roblox launching
- Cookie-based login
- Multi-instance support
- Private server link support
- Share link support
- Fishstrap compatibility
- Automatic Roblox version detection
- Roblox process monitoring

## Installation

```bash
npm install
npm start
```

## Build

```bash
build.bat
```

## Requirements

- Windows
- Node.js
- Roblox or Fishstrap installed
